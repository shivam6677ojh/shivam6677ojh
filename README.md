## 💡 What is Full Stack Development?

Full Stack Developers work on both:

- 🎨 **Front End**: UI/UX (what users see)
- ⚙️ **Back End**: Server, APIs, database logic

---

## 🧱 Tech Stack

| Area         | Technology                       |
|--------------|----------------------------------|
| Front End    | HTML, CSS, JavaScript, React     |
| Back End     | Node.js, Express.js              |
| Database     | MongoDB / MySQL                  |
| Tools        | Git, GitHub, Postman, VS Code    |
| Deployment   | Netlify, Vercel, Render          |

---

## 🛣️ Learning Roadmap

### 🧑‍🎓 1. Basics
- ✅ HTML & CSS
- ✅ JavaScript fundamentals
- ✅ Client-Server model

### ⚙️ 2. Frontend
- HTML + CSS layouts
- JavaScript interactivity
- React.js components

### 🔧 3. Backend
- Node.js runtime
- Express.js APIs

### 💾 4. Databases
- MongoDB (NoSQL)
- MySQL (SQL)

### 🔄 5. Integration
- REST APIs
- Fetch / Axios

### 🧪 6. Testing
- DevTools
- Postman
- Debugging

---

## 🚀 Deployment
- Frontend → Vercel / Netlify  
- Backend → Render / Heroku
